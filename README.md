# LRP-CCE-Lit-Review
This is a public repository for files related to the review paper "Leakage rate prediction through cracked concrete elements: A literature review". A PDF of this paper is available online [here](https://doi.org/10.1016/j.engstruct.2026.122855).
The authors of this paper gratefully appreciate the financial support of Conexus Nuclear Inc., who made the experimental program possible.

# Leakage Rate Data Workbook
Leakage rate data is available as a Microsoft Excel workbook file for download. This file name is "Leakage-Rate-Data-Workbook.xlsx". Updates to the leakage rate data will be made periodically and this updated data will be saved in workbooks that have different names.
